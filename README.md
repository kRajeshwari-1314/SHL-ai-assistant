🧠 SHL AI Assistant

An AI-powered assistant designed to help users with SHL assessment preparation, skill matching, and intelligent question/answer support using Python and machine learning concepts.

📌 Project Overview
The SHL AI Assistant is a smart system that helps users:
Understand assessment questions
Match skills with job requirements
Get AI-based suggestions and explanations
Improve preparation efficiency for SHL tests

⚙️ Tech Stack
Python 🐍
Machine Learning (if used)
NLP (Natural Language Processing)
FastAPI / Flask (if used in backend)
Pandas, NumPy
Git & GitHub

🚀 Features
📄 Resume / skill-based analysis
🤖 AI-powered response generation
🎯 Job-role skill matching
📊 Smart recommendations
🔍 Question understanding and explanation

🛠️ Installation & Setup
1. Clone repository
git clone https://github.com/kRajeshwari-1314/SHL-ai-assistant.git
2. Move into project folder
cd SHL-ai-assistant
3. Create virtual environment
python -m venv venv
4. Activate virtual environment
venv\Scripts\activate
5. Install dependencies
pip install -r requirements.txt
6. Run project
python app.py
